# Pokédex

Pokédex is a small web application that uses the PokéAPI to fetch, display, and explore Pokémon data.

## Live Demo

[Open live demo](https://osman-annak.developerakademie.net/modul-8/PokeAPI/index.html)

## Features

- Fetch Pokémon data from an external API
- Display Pokémon cards
- Show details for selected Pokémon
- Search or browse Pokémon
- Responsive design

## Tech Stack

- HTML
- CSS
- JavaScript
- REST API
- PokéAPI

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/OsmanAnk/PokeAPI.git
```

2. Navigate into the project folder:

```bash
cd PokeAPI
```

3. Open the project:

Open the `index.html` file in your browser or use the Live Server extension in VS Code.

> Note: This project fetches data from the PokéAPI, so an internet connection is required.
